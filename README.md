Objective
To analyze and visualize literacy and education disparities across Ghana using the 2021 Population and Housing Census (PHC) microdata.

Tools & Techniques
- Power BI (DAX, Power Query)
- Geospatial Analysis with Shapefiles
- Data Cleaning & Transformation
- Data Storytelling

Dashboard Pages
  1. National Overview: Key literacy indicators
  2. Gender Insights: Male vs Female literacy
  3. Regional Insights: Literacy across 16 regions

Key Insights
- National literacy rate: 69.6%  
- Male literacy: 73.9%, Female: 65.5% (8pp gap)  
- Urban literacy: 80%, Rural: 55%  
- Northern regions lag behind southern counterparts
